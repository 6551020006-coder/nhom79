# Game Raylib C++

Đây là một dự án game được phát triển bằng C++ sử dụng thư viện Raylib. Dự án được thiết lập để chạy trên Visual Studio Code trên Windows.

## Mô tả dự án

Dự án này là một game được xây dựng bằng C++ với sử dụng thư viện Raylib phiên bản 5.0. Game có các tính năng cơ bản với đồ họa 2D và tương tác chuột bàn phím. Dự án đã được kiểm tra trên Windows 10 và Windows 11.

## Công nghệ sử dụng

- C++ (90.2%)
- Python (7.6%)
- Batchfile (2.2%)
- Raylib 5.0

## Yêu cầu hệ thống

- Windows 10 hoặc Windows 11
- Visual Studio Code
- Raylib 5.0
- C++ Compiler

## Hướng dẫn cài đặt

1. Tải hoặc clone repository này về máy tính của bạn

2. Mở thư mục dự án trong Visual Studio Code

3. Mở file main.code-workspace bằng cách double click vào nó trong File Explorer

4. Navigateto thư mục src trong Explorer Window của VS Code

5. Double click vào file main.cpp để mở nó

6. Cài đặt các extension cần thiết cho C++ trong VS Code nếu chưa có

## Hướng dẫn chạy chương trình

1. Mở file main.cpp trong src folder

2. Nhấn F5 trên bàn phím để biên dịch và chạy chương trình

3. Game sẽ khởi động và hiển thị cửa sổ game

## Cấu trúc thư mục

- src/: Chứa tất cả source code của game
  - main.cpp: File chính của chương trình
- include/: Chứa các file header
- lib/: Chứa thư viện Raylib
- main.code-workspace: File cấu hình cho VS Code

## Hướng dẫn chơi game

Game chứa một máy bay di chuyển trên màn hình. Bạn có thể tương tác với game bằng chuột hoặc bàn phím.

## Tính năng chính

- Đồ họa 2D với Raylib
- Tương tác chuột và bàn phím
- Vòng lặp game hoạt động mượt mà
- Dễ dàng mở rộng và chỉnh sửa
